üìã The Rapid Acquisition Protocol

A repeatable 6-step process for mastering any codebase quickly:

1 Initial Reconnaissance

Feed AI the codebase. Ask for structure, not details. Understand the shape before the substance.

2 Entry Point Mapping

Where does execution start? API routes, CLI commands, job schedulers. These are your map's landmarks.

3 Data Model Analysis

What are the nouns? Database schemas, domain objects, API types. Understanding data = understanding the system.

4 Pattern Recognition

How do they do things? Naming conventions, error handling, testing approaches. Learn the local dialect.

5 Risk Assessment

Where are the landmines? Complex areas, tech debt, missing tests, security concerns.

6 Documentation Generation

What's missing? Have AI draft docs you wish existed. Then verify and refine.
