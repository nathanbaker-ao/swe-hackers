Part 2: Direct AI to Build

ü§ñ

Work with Cursor

  1. Choose one small feature
    E.g., "a function to validate user input" or "a class for a Post object"
  2. Write a clear prompt
    Describe what you want in plain English. Reference your diagrams.
  3. Press Cmd+K in Cursor
    Give your prompt to the AI and let it generate code.
  4. Review the output
    Read every line. Understand it. Ask AI to explain anything unclear.
  5. Iterate if needed
    "Change this to use a dict instead of a list" ‚Äî refine until it's right.
