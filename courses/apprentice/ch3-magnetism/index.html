Imagine you have a magic machine. You put something in, it does its magic, and something new comes out. You don't need to know how the machine works inside â€” you just need to know what to put in and what you'll get out.

That's what functions are! They're reusable machines that take inputs, do something with them, and give you back outputs. Functions let you write code once and use it many times.

ðŸ§  Why Functions Matter

Without functions, you'd have to copy-paste the same code everywhere. With functions, you write it once and call it whenever you need it. This is called DRY: Don't Repeat Yourself!

ðŸ”§ Creating Functions

In Python, you create a function using the def keyword (short for "define"):

Python 
# Define a function

    print("Hello, world!")

# Call the function (use it!)
say_hello()  # Output: Hello, world!
say_hello()  # Output: Hello, world!
say_hello()  # Output: Hello, world!

ðŸ’¡ Key idea: Define the function once, then call it as many times as you want! Notice the parentheses () â€” they're required.

ðŸ“¥ Parameters: Giving Functions Data

Functions become powerful when you can give them inputs. These inputs are called parameters.

Python 
# Function with one parameter
def greet(name):
    print(f"Hello, {name}!")

greet("Alex")    # Output: Hello, Alex!
greet("Jordan")  # Output: Hello, Jordan!
greet("Sam")     # Output: Hello, Sam!

Multiple Parameters

Python 
# Function with two parameters
def add_numbers(a, b):
    result = a + b
    print(f"{a} + {b} = {result}")

add_numbers(5, 3)    # Output: 5 + 3 = 8
add_numbers(10, 20)  # Output: 10 + 20 = 30

ðŸ’¡ Think of it like: The function is a recipe. Parameters are the ingredients. You can use the same recipe with different ingredients!

def ask_question(question, correct_answer):
    """Ask a question and check if the answer is correct."""
    answer = input(question + " ")
    if answer.lower() == correct_answer.lower():
        print("âœ“ Correct!")
        return True
    else:
        print(f"âœ— Wrong! The answer was: {correct_answer}")
        return False

def calculate_score(correct, total):
    """Calculate the percentage score."""
    return (correct / total) * 100

# Main quiz
print("ðŸ§  QUIZ TIME! ðŸ§ \n")

score = 0
questions = [
    ("What is the capital of France?", "Paris"),
    ("What is 7 x 8?", "56"),
    ("What color is the sky?", "blue"),
]

for i, (question, answer) in enumerate(questions, 1):
    print(f"Question {i}:")
    if ask_question(question, answer):
        score += 1
    print()
